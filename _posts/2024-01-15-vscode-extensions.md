개발 환경을 구축할 때 가장 먼저 고민하게 되는 것이 바로 에디터 선택이다. VS Code의 필수 확장 프로그램들을 소개한다.

## 왜 VS Code인가?

VS Code의 가장 큰 장점은 바로 풍부한 확장 프로그램 생태계다. 마이크로소프트가 오픈소스로 개발하고 있으며, 전 세계 개발자들이 자신만의 확장 프로그램을 만들어 공유하고 있다.

> 좋은 도구는 개발자의 생산성을 10배 이상 높여준다.

## 추천 확장 프로그램

### 1. Prettier - Code Formatter

코드 포매팅은 협업에서 필수적인 요소다. Prettier는 자동으로 코드 스타일을 일관되게 맞춰준다.

```json
{
  "editor.formatOnSave": true,
  "editor.defaultFormatter": "esbenp.prettier-vscode"
}
```

### 2. GitLens

Git 작업을 하다 보면 "이 코드는 누가 언제 작성했지?"라는 궁금증이 생긴다. GitLens는 각 라인별로 커밋 정보를 보여준다.

### 3. Thunder Client

API 테스트를 위해 Postman을 따로 실행할 필요 없이, VS Code 내에서 바로 REST API를 테스트할 수 있다.

## 마치며

좋은 도구를 찾는 것은 개발자로서 중요한 투자다. 자신의 워크플로우에 맞는 것들을 선별해서 사용하길 바란다.
