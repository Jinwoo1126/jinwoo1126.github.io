## 1. 개요 및 주요 기능

Figma MCP는 디자인 파일의 메타데이터를 읽고, 편집하고, 코드로 변환하는 과정을 AI가 직접 수행할 수 있도록 돕는 도구입니다.

- **정방향 엔지니어링**: Figma 디자인 → 코드 생성 (React, Tailwind CSS 등)
    
- **역방향 엔지니어링**: 기존 웹 서비스(URL) → Figma 디자인 레이어로 변환
    
- **다이어그램 생성**: Mermaid 구문을 사용하여 FigJam 다이어그램 자동 생성
    

---

## 2. 주요 도구(Tools) 및 기능 상세

Figma MCP에서 제공하는 핵심 API 도구들은 다음과 같습니다.

|**도구명**|**기능 설명**|
|---|---|
|**`get_design_context`**|특정 노드나 파일의 상세 메타데이터를 획득 (코드 생성 시 필수)|
|**`Youtube`**|파일의 전체적인 구조와 노드 목록을 빠르게 파악|
|**`get_screenshot`**|특정 노드의 시각적 참조 이미지를 확보하여 AI가 디자인을 '보게' 함|
|**`generate_figma_design`**|**핵심 기능!** 웹앱 UI나 URL을 Figma의 편집 가능한 레이어로 역변환|
|**`generate_diagram`**|Mermaid 구문을 기반으로 FigJam 내 다이어그램 생성|
|**`get_variable_defs`**|디자인 토큰(색상, 간격 등 변수 정의) 확인|
|**`whoami`**|현재 인증된 사용자 정보 및 남은 쿼터(Quota) 확인|

---

## 3. 권장 워크플로우 (디자인 → 코드 변환 시)

AI에게 디자인을 분석하고 코드를 짜달라고 할 때, 다음 순서로 도구를 호출하는 것이 효율적입니다.

1. **`get_design_context`**: 전체적인 디자인 컨텍스트 파악
    
2. **`Youtube`**: (응답이 클 경우) 필요한 특정 노드만 필터링하여 다시 파악
    
3. **`get_screenshot`**: 시각적 참조 이미지 확보 (정확도 향상)
    
4. **`get_variable_defs`**: 디자인 토큰(변수)을 확인하여 코드에 반영
    
5. **코드 생성 시작**: 위 정보를 바탕으로 React 컴포넌트 등 작성

## 4. 역방향 워크플로우: `generate_figma_design`

이미 운영 중인 웹사이트를 Figma 파일로 옮기고 싶을 때 사용하는 강력한 기능입니다.

- **작동 방식**: URL 입력 → 브라우저 캡처 스크립트 실행 → 레이어 구조 분석 → Figma 파일 생성
    
- **출력 모드(outputMode)**:
    
    - `newFile`: 새로운 Figma 파일 생성 (기본값)
        
    - `existingFile`: 기존 파일 내 특정 페이지에 추가
        
    - `clipboard`: 클립보드에 복사하여 Figma에서 직접 붙여넣기
        

> [!WARNING] **플랜 주의사항**
> 
> Starter 플랜의 경우 호출 제한이 엄격합니다. 한 번의 캡처 과정에서 `whoami`와 `generate_figma_design` 등이 여러 번 호출되므로, 월 6회 한도가 금방 소진될 수 있습니다.

## 5. 실습 예시: Next.js 프로젝트 역캡처

저자는 실제 본인의 사이트(`goddaehee.co.kr`)나 Next.js로 만든 프로젝트를 예시로 듭니다.

1. **단계 1**: 실행 중인 로컬호스트나 외부 URL을 `generate_figma_design`으로 캡처
    
2. **단계 2**: 생성된 Figma 레이어 확인 (단순 이미지가 아닌 수정 가능한 컴포넌트 구조)
    
3. **단계 3**: 해당 Figma 결과물을 다시 AI에게 주어 코드로 변환 (양방향 흐름 완성)

## 6. 요약 및 결론

- **Figma MCP**는 디자이너와 개발자 사이의 간극을 줄여주는 혁신적인 도구입니다.
    
- 단순히 디자인을 보는 것을 넘어, **웹을 디자인으로, 디자인을 코드로** 자유롭게 변환할 수 있습니다.
    
- **Claude Code**와 같은 터미널 기반 AI와 결합했을 때 가장 강력한 생산성을 보여줍니다.
